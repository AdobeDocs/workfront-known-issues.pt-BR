# Workfront Proof: as provas não aparecem nas pastas


>[!NOTE]
>
>Esse problema foi corrigido em 23 de fevereiro de 2026.


Quando um usuário visualiza uma pasta que deve conter ativos, a pasta não exibe ativos.
