# Usuários: problemas com alterações no perfil de prova

>[!NOTE]
>
>Esse problema foi corrigido em 17 de dezembro de 2025.

Os seguintes problemas foram relatados em relação a alterações no perfil de prova de um usuário:

* Quando um usuário atribui um perfil de prova diferente a um usuário e o salva, a alteração não é salva, e o usuário mantém o perfil de prova anterior.
* O perfil de prova de um usuário pode ser exibido como o perfil de prova incorreto, mesmo que o perfil de prova correto tenha sido atribuído.
* Quando outras alterações são feitas em um usuário, as permissões de prova também podem mudar.

_Relatado pela primeira vez em 24 de novembro de 2025._
