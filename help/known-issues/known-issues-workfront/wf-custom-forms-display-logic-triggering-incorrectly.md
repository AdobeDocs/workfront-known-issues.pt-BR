# Formulários personalizados: a lógica de exibição de campos não é seguida corretamente

>[!NOTE]
>
>Esse problema foi corrigido em 22 de outubro de 2025.

Quando um usuário está preenchendo um formulário personalizado que inclui lógica de exibição (em que alguns campos são exibidos ou não com base nos valores de outros campos), os campos não são exibidos de uma maneira que siga essa lógica de campo.

_Relatado pela primeira vez em 6 de outubro de 2025._
