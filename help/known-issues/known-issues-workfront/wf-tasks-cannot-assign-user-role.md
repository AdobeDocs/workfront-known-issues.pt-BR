# Tarefas: não é possível selecionar a função de usuário ao criar uma tarefa

Quando um usuário cria uma tarefa e tenta atribuir uma função de usuário, a opção para atribuir a função não está presente e o usuário não pode atribuir uma função de usuário.

**Solução alternativa**

Crie a tarefa e edite-a para atribuir a função de usuário em Atribuições Avançadas.

_Relatado pela primeira vez em sábado, 28 de novembro de 2025._
