# Tarefas: não é possível selecionar a função de usuário ao criar uma tarefa

>[!NOTE]
>
>Esse problema foi corrigido em 21 de dezembro de 2025.

Quando um usuário cria uma tarefa e tenta atribuir uma função de usuário, a opção de atribuir a função não está presente, e o usuário não pode atribuir uma função de usuário.

**Solução**

Crie a tarefa e edite-a para atribuir a função de usuário em “Atribuições avançadas”.

_Relatado pela primeira vez em 28 de novembro de 2025._
