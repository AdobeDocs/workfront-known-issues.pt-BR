# Grupos: não é possível adicionar um subgrupo

Quando um usuário tenta adicionar um subgrupo existente a um grupo, o botão Salvar não é eficaz e o subgrupo não é adicionado.

_Relatado pela primeira vez em quinta-feira, 1 de maio de 2024._
