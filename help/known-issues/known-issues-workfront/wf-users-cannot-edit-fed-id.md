# Usuários: não é possível editar a Federation ID de um usuário

>[!NOTE]
>
>* Esse problema foi corrigido em 23 de setembro de 2025.
>* Esse problema afeta organizações que ainda não foram migradas para o sistema de gerenciamento de identidades da Adobe

Quando um administrador tenta atualizar a Federation ID de um usuário, o campo referente a essa informação não está disponível, e o administrador não pode atualizá-lo.

Isso significa que o usuário não consegue fazer logon no Workfront.

_Relatado pela primeira vez em 9 de setembro de 2025._
