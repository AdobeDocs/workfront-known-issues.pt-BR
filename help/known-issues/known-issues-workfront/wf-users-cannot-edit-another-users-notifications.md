# Usuários: não é possível editar configurações de notificações para outro usuário

>[!NOTE]
>
>Esse problema foi corrigido em 25 de setembro de 2025.

Quando um usuário tenta editar as configurações de notificação de outro, ele não consegue e vê a seguinte mensagem:

“Você não tem acesso suficiente para editar dados financeiros”.

_Relatado pela primeira vez em 9 de setembro de 2025._
