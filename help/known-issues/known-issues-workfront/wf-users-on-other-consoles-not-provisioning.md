# Usuários: os usuários não são provisionados corretamente na Adobe Admin Console

Se um usuário já existir em uma Adobe Admin Console e for provisionado automaticamente em um Console que inclua o Workfront, ele não será provisionado corretamente para o novo Console.

_Relatado pela primeira vez em sexta-feira, 20 de novembro de 2025._
