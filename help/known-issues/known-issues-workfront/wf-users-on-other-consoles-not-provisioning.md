# Usuários: os usuários não são provisionados corretamente no Adobe Admin Console

>[!NOTE]
>
>Esse problema foi corrigido em 3 de fevereiro de 2026.

Se um usuário já existir em um Adobe Admin Console e for provisionado automaticamente em um Console que inclua o Workfront, ele não será provisionado corretamente para o novo Console.

_Relatado pela primeira vez em 20 de novembro de 2025._
