# Usuários: níveis de acesso mais baixos não disponíveis ao criar um usuário

Quando um usuário cria outro usuário, somente o nível de acesso do primeiro usuário está disponível para o usuário recém-criado. Todos os níveis de acesso com permissões mais baixas que as do usuário que está criando devem estar disponíveis para atribuir ao novo usuário.

_Relatado pela primeira vez em quarta-feira, 30 de janeiro de 2024._
