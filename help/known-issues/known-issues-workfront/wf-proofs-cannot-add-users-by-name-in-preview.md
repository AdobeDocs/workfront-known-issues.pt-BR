# Provas: não é possível adicionar usuários a uma prova por nome no ambiente de pré-visualização

>[!NOTE]
>
>Esse problema foi relatado no ambiente de sandbox de pré-visualização

Quando um usuário tenta adicionar outro usuário a uma prova e procura esse usuário pelo nome, o nome não é exibido e, assim, o usuário não consegue selecioná-lo.

Isso pode ocorrer mesmo após a atualização do ambiente de sandbox de pré-visualização.

_Relatado pela primeira vez em 5 de maio de 2025._
