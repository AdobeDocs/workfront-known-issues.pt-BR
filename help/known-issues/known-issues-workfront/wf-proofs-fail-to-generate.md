# Provas: provas não são geradas

<!--Wf and WFP TOCs-->

>[!NOTE]
>
>Esse problema foi corrigido em 24 de agosto de 2023.

Quando você tenta criar uma prova de um documento enviado para o [!DNL Workfront], a prova não é gerada.
