# Usuários: não é possível atualizar o acesso do usuário

Quando um usuário tenta atualizar o acesso de outro, de “Visualizar” para “Gerenciar”, em um projeto, ele não consegue e vê a seguinte mensagem:

“Você não tem acesso suficiente para visualizar as finanças deste projeto”.

_Relatado pela primeira vez em 11 de dezembro de 2025._
