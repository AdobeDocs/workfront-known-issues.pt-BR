# Usuários: não é possível atualizar o acesso do usuário

Quando um usuário tenta atualizar o acesso de outro usuário da Exibição para Gerenciar em um projeto, ele não pode atualizar o acesso e vê a seguinte mensagem:

&quot;Você não tem acesso suficiente para visualizar o financiamento deste projeto.&quot;

_Relatado pela primeira vez em sexta-feira, 11 de dezembro de 2025._
