# Usuários: os usuários removidos do Adobe Admin Console ainda estão ativos no Workfront

Quando um administrador remove um usuário do Adobe Admin Console, esse usuário não é removido do Workfront. O usuário ainda está visível e ativo no Workfront.

_Relatado pela primeira vez em quarta-feira, 3 de fevereiro de 2026._
