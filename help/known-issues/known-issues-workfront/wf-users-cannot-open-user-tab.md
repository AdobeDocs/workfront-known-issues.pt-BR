# Usuários: erro ao abrir a página “Usuários”

>[!NOTE]
>
>Esse problema foi corrigido em 12 de dezembro de 2025.

Quando um usuário tenta abrir a página “Usuários” a partir do menu principal, a página não abre, e o usuário vê a seguinte mensagem:

“Ocorreu um erro e estamos trabalhando para resolver o problema. Para continuar com o seu trabalho, tente atualizar esta página do navegador”.

_Relatado pela primeira vez em 8 de dezembro de 2025._
