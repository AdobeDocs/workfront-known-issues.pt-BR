# Integrações: as integrações do JumpSeat falham para organizações no Unified Shell

>[!NOTE]
>
>Esse problema foi resolvido, mas exige que os usuários reconfigurem o JumpSeat conforme descrito na **Solução** abaixo.

Quando a organização de um usuário é migrada para o Adobe UnifiedBecky Shell, suas integrações já existentes com o JumpSeat não funcionam.

**Solução alternativa**

No painel do administrador do JumpSeat, altere o URL do aplicativo de `workfront.com` para `workfront.adobe.com`.

_Relatado pela primeira vez em 30 de outubro de 2024._
