# Solicitações: a visualização &quot;Tudo&quot; não mostra todas as solicitações

>[!NOTE]
>
>Este problema foi relatado na nova experiência de solicitação

Quando um usuário exibe a lista de solicitações na nova experiência de solicitação e a exibição está definida para mostrar todas as solicitações, somente algumas das solicitações são exibidas.

_Relatado pela primeira vez em quarta-feira, 16 de dezembro de 2025._
