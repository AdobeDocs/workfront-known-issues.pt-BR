# Formulários personalizados: não é possível editar formulários personalizados devido ao campo de Texto descritivo

>[!NOTE]
>
> Esse problema foi corrigido em 11 de agosto de 2022.

Quando um usuário tenta editar um formulário personalizado que inclui um campo de [!UICONTROL Texto descritivo], o rótulo do campo não é preenchido. O usuário vê o erro “[!UICONTROL Este campo é obrigatório]” no campo de rótulo e não consegue editar o formulário personalizado devido a esse erro.

_Reportado pela primeira vez em 14 de julho de 2022._
