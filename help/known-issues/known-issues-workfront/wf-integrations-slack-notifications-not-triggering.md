# Integrações: as notificações do Slack da Workfront não são acionadas

Quando ocorre um evento no Workfront que deve acionar uma notificação do Slack, a notificação não é enviada.

Esse problema foi relatado para os seguintes eventos:

* Um usuário comenta em uma tarefa ou projeto atribuído ao usuário afetado
* O usuário afetado é mencionado ou marcado em um comentário no Workfront.

Isso pode ocorrer mesmo que as preferências de notificação do usuário estejam configuradas para enviar a notificação no Slack.

_Relatado pela primeira vez em terça-feira, 2 de fevereiro de 2026._
