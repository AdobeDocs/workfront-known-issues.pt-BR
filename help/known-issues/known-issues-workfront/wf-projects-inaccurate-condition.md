# Projetos: o status do progresso do projeto não é atualizado automaticamente para “Atrasado”

>[!NOTE]
>
>Esse problema foi encerrado porque o funcionamento está correto conforme projetado.

Quando o status do progresso de um projeto deve ser atualizado automaticamente para “Atrasado”, como quando a data de conclusão planejada passa e o projeto não é concluído, o status do progresso não é atualizado. Isso também significa que a condição do projeto não é movida para “Com problema”.

_Relatado pela primeira vez em 13 de março de 2025._
