# Formulários personalizados: não é possível editar campos de rich text

>[!NOTE]
>
>Esse problema foi corrigido em terça-feira, 22 de dezembro de 2025.

Quando um usuário tenta editar um campo de rich text em um formulário personalizado, as alterações não são salvas e o campo não é atualizado para refletir as alterações.

_Relatado pela primeira vez em quinta-feira, 3 de dezembro de 2025._
