# Tarefas: as edições na seção Atribuições de uma tarefa não são salvas

>[!NOTE]
>
>Esse problema se aplica à experiência de tarefa herdada.

Quando um usuário faz alterações na seção Edições de uma tarefa na caixa de diálogo de tarefa herdada e as salva, as alterações não são salvas.

**Solução alternativa**

Para algumas alterações, você pode usar a nova experiência de tarefa.

_Relatado pela primeira vez em terça-feira, 24 de novembro de 2025._
