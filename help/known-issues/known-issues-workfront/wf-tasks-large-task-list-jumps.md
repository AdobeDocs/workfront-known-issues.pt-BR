# Tarefas: uma grande lista de tarefas salta inesperadamente para um local diferente na lista

Quando um usuário está visualizando uma grande lista de tarefas e atualiza uma tarefa ou expande uma tarefa pai, a lista de tarefas salta inesperadamente para um local diferente na lista.

_Reportado pela primeira vez em 22 de janeiro de 2026._
